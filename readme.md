Agent Luhmann
===

AI-powered Zettelkasten for Obsidian.

## Run

```bash
npm install
npm run dev          # watch mode
npm run build        # production
```

Copy `main.js`, `manifest.json`, and `styles.css` (if any) to:

```
<Vault>/.obsidian/plugins/agent-luhmann/
```

Reload Obsidian and enable the plugin in **Settings → Community plugins**.

## The Features

Obsidian plugin ("Agent Luhmann") that codifies the Zettelkasten

### Capture & Filter

The capture-and-filter workflow — specifically the zero-friction capture of **fleeting notes** and their subsequent evaluation through a **surprise** rating system, governed by the principle that note value equals surprise multiplied by connectivity.

- **Surprise as Filter** — Prioritize notes that shift your understanding; discard the predictable
